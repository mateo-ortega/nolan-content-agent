Diagnóstico, prioridades y roadmap de 5 tiers para llevar a Nolan del estado funcional actual al sistema que aprende, escala y se replica.

Fecha: 18 mayo 2026 Estado: funcional end-to-end Próximo bloqueo crítico: feedback loop ciego

Resumen ejecutivo

  • Nolan produce carrusel, animación Manim y talking-head + post-pro en pipeline cerrado, con aprobación humana, ética codificada y budget guard. La base es sólida.
  • El cuello de botella no es técnico, es de información: el sistema no ve qué pasa después de publicar. Sin métricas de IG, el learning loop es ciego al 95% del flujo.
  • La prioridad #1 es cerrar el feedback loop (Tier 1). Después: voiceover end-to-end y hardening. Multi-tenant solo si se decide vender el sistema a clientes externos.

02 · roadmap priorizado

5 tiers, 8-12 semanas, impacto medible

01

Cerrar el feedback loop

crítico 2-3 semanas
Sin métricas de engagement, el learning loop ve solo el 5% del flujo (los rechazos). Esto bloquea cualquier mejora real de calidad porque Nolan no sabe qué funciona.
  • Skill nolan-analytics: Apify IG scraper para @sapiens.ed, tabla pieces_metrics en SQLite, cron semanal.
  • Extender rule_writer.py: detectar patrones de top 20% engagement (no solo rechazos), generar reglas tanto "evitar X" como "repetir Y".
  • Dashboard Telegram semanal: top 3 piezas, bottom 3, costo total, presupuesto restante, tasa de aprobación.
Qué desbloquea Learning loop bidireccional. Permite contestar "¿qué formato/nicho/CTA está funcionando?" con datos reales, no intuición.
Impacto Crítico
Esfuerzo Medio
Costo extra ~$5/mes
02

Voiceover end-to-end

alto impacto 2 semanas
El formato de mayor engagement en IG hoy (reels con voiceover + b-roll) está bloqueado únicamente por la reference recording. Es el último formato faltante del mix prometido.
  • Reference recording Mateo: 30 min en condiciones controladas → ElevenLabs voice clone "Mateo Sapiens".
  • Skill nolan-produce-voiceover: Pexels/Pixabay search por keywords → crop a 9:16 → ElevenLabs TTS → ffmpeg composite + subs (reutilizar reels-postpro/subtitles.py) + music ducking.
  • Integración al ciclo: martes/jueves alternar con animación; regla en decide-format "tema narrativo emocional sin matemática → voiceover".
Qué desbloquea Mix completo de 4 formatos. 5 piezas/semana sostenibles. Capacidad de cubrir nichos emocionales (padres) sin depender de Mateo grabándose.
Impacto Alto
Esfuerzo Medio-alto
Costo extra ~$15/mes ElevenLabs
03

Hardening y reproducibilidad

medio impacto 1-2 semanas
Si el VPS muere, el aprendizaje histórico se pierde. Si quieres replicar a otro cliente, no hay procedimiento. Trabajo de plomería pero hoy es un riesgo silencioso.
  • Backup SQLite automático: scripts/backup_sqlite_cron.sh con rclone copy a Drive diario 03:00, retención 30 días.
  • Disaster recovery: docs/disaster_recovery.md con pasos de redeploy desde repo + Hostinger snapshot. Probar restore en VPS scratch 1 vez.
  • Smoke test full pipeline: tests/smoke_test_full_pipeline.sh ejecuta research → decide → produce → package en --dry-run, valida dimensiones, costo, ethics.
  • Fix produce_guion.py: remover emoji + hashtags, integrar EthicsGate, añadir prompt caching.
  • Actualizar HANDOFF.md: fecha real, gaps reales, próximos pasos.
Qué desbloquea Sistema robusto ante fallas. Calidad uniforme entre producers. Documentación de handoff confiable. Prerequisito para vender a externos.
Impacto Medio
Esfuerzo Bajo-medio
Costo extra $0
04

Multi-tenant / preparación clientes externos

condicional 2 semanas
Solo se ejecuta si la decisión es vender Nolan como servicio de agencia. Sin esto, replicar a un cliente nuevo toma 1+ semana de trabajo manual.
  • Parametrizar reels-postpro: mover SAPIENS_TEAL_HEX y constantes de subtitles.py/audio.py a brand_config.yaml por cliente.
  • Onboarding wizard: scripts/onboard_client.sh con 8 preguntas → genera clients/<name>/ con SOUL.md custom, brand_context.md, colores, fonts, nichos, .env, rclone path.
  • Tiempo target setup cliente: <4 horas.
  • Doc comercial: docs/client_onboarding.md con checklist 1 semana, docs/pricing_template.md con tiers.
Qué desbloquea Vender Nolan como servicio gestionado con setup fee + retainer. Replicabilidad real, no copy-paste manual.
Impacto Condicional
Esfuerzo Medio
Costo extra $0
05

Calidad y escala (futuro)

opcional 2-3 semanas
Optimización fina. No bloqueante, pero diferencia un "agente de contenido" de un "sistema de marketing inteligente". Solo después de Tier 1-3 sólidos.
  • A/B testing de variantes: --variants=2 genera hook A vs B, CTA A vs B. Telegram approval elige o publica ambas. pieces.sqlite trackea ganadora.
  • Multi-canal LinkedIn: skill nolan-linkedin reescribe copy a 1200×1200 y adapta tono.
  • Self-improvement de prompts: cuando una regla se valida con >50 piezas, propone editar el prompt base del producer, no solo SOUL.md. Versionado git-style.
  • Observabilidad: dashboard HTML estático con tasa de aprobación por nicho/formato/semana.
Qué desbloquea Optimización iterativa autónoma. Cobertura multi-plataforma. Sistema que mejora sin intervención humana en el prompt engineering.
Impacto Largo plazo
Esfuerzo Alto
Costo extra $5-20/mes
